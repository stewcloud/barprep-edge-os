# BarPrep Edge v0.3.1

This package directly patches the current `barprep-edge-os` source and reinstalls it.

## Install

```bash
unzip barprep-edge-v0-3-1.zip
cd barprep-edge-v0-3-1
sudo bash install.sh ~/barprep-edge-os
```

Afterward, confirm the dashboard reports `0.3.1`. The pairing page must ask only
for the BarPrep Core URL. It must not contain an alphanumeric pairing-code field.
